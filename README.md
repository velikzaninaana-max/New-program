## Информация о проекте :crystal_ball:

**New program** — это веб-приложение, которое позволяет *анализировать* и *визуализировать* активность пользователя или команды в GitHub-репозиториях. 
Проект помогает понять динамику разработки, выявить ключевые изменения, частоту коммитов и вклад участников. 

:heavy_exclamation_mark: Проект еще находится в стадии **активной разработки**, поэтому ~~могут быть ошибки~~.

### Необходимые условия

Для запуска проекта необходимы следующие компоненты:

|Компонент|Версия|Описание|
|:-|:-:|:-:|
|Python|3.8+|Основной язык программирования|
|Git|2.20+|Система контроля версий|

### Установка

1. **Клонируйте репозиторий** на ваш локальный компьютер:

    ``git clone https://github.com/velikzaninaana-max/New-program.git``

2. **Перейдите в директорию проекта**:

    ``cd New-program/src/New-program``
